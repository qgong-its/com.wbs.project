# Figma to website

![Status: Archived](https://img.shields.io/badge/Status-Archived-red) 

**Notice: Project Completed & Archived**

A simple semantic HTML/CSS website built as part of our bootcamp introduction project.

The project demonstrates basic layout, semantic HTML structure, and collaboration using GitHub workflow.

Development Workflow

The project was developed using a simple collaborative workflow:

1. Tasks were organized in Trello using a Kanban board.
2. The page layout was first designed as a wireframe in Figma.
3. Features were implemented in separate Git branches.
4. Changes were merged into the `main` branch using Pull Requests.
5. The final site is deployed via GitHub Pages.

## Live Demo

https://qgong-its.github.io/com.wbs.figma-to-website/index.html

## Features

- Semantic HTML structure
- Hybrid CSS Layout:
  - Flex utilized for the navigation header and alignment of single-axis elements.
  - Grid implemented for e.g. the "Why choose us" section to achieve a responsive x\*y feature grid as per the wireframe.
- Responsive layout (optional)
- Deployed via GitHub Pages
